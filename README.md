# Note Taker Starter Code

## Description
This app will allow the user to write and save notes to assist with task-tracking and the organizing of one's thoughts.

### Languages used:
- HTML
- CSS
- JavaScript
- Node
- ES6

1. npm init -y
1. Enter "node server" in the command line to run.
